## 香橙派R1 Plus LTS OpenWRT固件
    采用LEDE源码编译
    
    集成大部分常用插件（Docker）
    
    添加官方PWM风扇脚本
    
    kernrl_6.1.20及以后版本修复朗科、川宇等SD卡软重启一直亮红灯死机问题
    
    添加官方YT8531C网卡驱动
    
    默认密码password
    
    Action不再维护
    
    有部分板子体质差，请自行测试，估计绷不住
    
    下载固件请去release
